## About

I am an Indonesia-based researcher and developer working at the intersection of science education, educational technology, and practical software systems.

My current work focuses on doctoral research in Science Education at Universitas Negeri Padang, while also building digital tools that support academic data, documentation, learning resources, and institutional workflows.

I care about systems that are clear, useful, well-documented, and reliable enough to be used by real people.

## Current Focus

- Doctoral research in Science Education
- Educational technology and learning innovation
- Web applications for academic and campus workflows
- AI-assisted tools for research, documentation, and learning support
- Clean documentation, reproducible workflows, and maintainable code
